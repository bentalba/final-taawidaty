# Blog Images

This directory contains featured images for blog posts.

## Required Images (1200x630px):

1. **cnss-guide.jpg** - Featured image for CNSS reimbursement guide
2. **cnops-guide.jpg** - Featured image for CNOPS reimbursement guide
3. **cnss-vs-cnops.jpg** - Featured image for CNSS vs CNOPS comparison

## Design Guidelines:

- Dimensions: 1200 x 630 pixels (Open Graph standard)
- Format: JPG (optimized for web)
- Include: Taawidaty logo, article title, relevant medical/healthcare imagery
- Colors: Match brand colors (primary blues/greens)
- Text: Large, readable font

## How to Create:

Use Canva or Photoshop with the template "Blog Banner" or "Social Media Post"

## Temporary Placeholder:

Until custom images are created, you can use a solid color placeholder or the default Taawidaty logo.
